## 这不是另一份使用手册

官方文档已经准确回答“如何安装、配置和使用 Grok Build”。本站换一个观察方向：从公开 Rust 源码追踪输入如何变成 `Turn`，模型事件如何落入 TUI，工具调用如何越过权限与沙箱边界，以及状态如何被持久化、压缩和恢复。

每篇文章都锁定同一个公开提交，并把关键判断连接到精确行号。源码没有直接表达的设计意图会标注为**推断**；无法由公开代码证明的产品行为不会写成事实。

## 源码与官方文档的分工

| 你想解决的问题 | 首选入口 |
| --- | --- |
| 安装、命令、配置项、扩展用法 | [Grok Build 官方文档](https://docs.x.ai/build/overview) |
| 类型之间如何协作、事件怎样流动 | 本站架构文章 |
| 某项行为在当前版本是否真实存在 | 本站固定行号 + 上游源码 |
| 最新版本是否已经变化 | 上游主分支与官方发布说明 |

本站是非官方社区学习项目，不代表 xAI；所有源码链接均指向固定快照，而不是会漂移的 `main` 分支。
